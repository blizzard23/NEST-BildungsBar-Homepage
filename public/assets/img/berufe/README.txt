BERUFS-BILDER
=============

Lege hier die Fotos für die einzelnen Berufe ab (z. B. pflegefachkraft.jpg).

Verknüpfe ein Bild mit einem Beruf in assets/berufe-data.js, indem du
den Beruf von einem reinen Text-Eintrag in ein Objekt mit "bild" änderst:

    "Pflegefachkraft"

wird zu:

    { name: "Pflegefachkraft", bild: "assets/img/berufe/pflegefachkraft.jpg" }

Empfohlen:
- Querformat, ca. 800 x 500 px (16:10), JPG/WebP
- Dateinamen klein und ohne Leerzeichen/Umlaute

Ist kein Bild hinterlegt, zeigt die Seite automatisch einen farbigen
Platzhalter mit dem Kategorie-Icon (jeder Beruf bekommt eine eigene Farbe).
